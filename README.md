# Python Number Guessing Game

A clean, interactive command-line tool built with Python that challenges users to guess a randomly generated number. This project demonstrates core programming logic, user input handling, and error validation.

## 🚀 Features
* **Dynamic Range:** Generates a random integer between 1 and 100.
* **Input Validation:** Handles non-numeric inputs gracefully using `try/except` blocks to prevent crashes.
* **Attempt Tracker:** Counts and displays the total number of guesses upon success.
* **Interactive Feedback:** Provides real-time "Higher" or "Lower" hints to guide the player.

## 🛠️ Technical Overview
The script utilizes Python's built-in `random` module and follows professional coding standards, including:
* **Modular Design:** Logic is encapsulated within a `start_game()` function.
* **Entry Point Protection:** Uses `if __name__ == "__main__":` to ensure the script runs correctly when executed.
